## üöÄ Getting Started

1. **Clone Repository**
   ```bash

   cd collective-memory/collective-memory-app
   ```

   ```bash

   npm install  # for frontend
   ```

3. **Start Smart Context Bridge**
   ```bash
   python src/context_bridge_cli.py start
   ```

   ```bash
   # In Cursor or any chat interface:
   ```
